# @opencodehub/pack

Deterministic code-pack generator producing the M5 9-item BOM (manifest, skeleton, file-tree, deps, ast-chunks, xrefs, embeddings-sidecar, findings, licenses). Scaffolded in AC-M5-1; BOM body implementations land in AC-M5-3..9. See `.erpaval/specs/005-m5-m6/spec.md` for the contract.
